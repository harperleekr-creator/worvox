# WorVox 월별 AI 및 서비스 비용 분석

## 1️⃣ OpenAI API 비용

### 사용 중인 모델:

#### **GPT-4o-mini** (메인 대화 모델)
- **사용처**: AI 대화, 단어 학습, AI 프롬프트 생성, 문법 분석
- **가격**:
  - Input: $0.150 / 1M tokens
  - Output: $0.600 / 1M tokens
- **예상 사용량** (월 1,000명 활성 유저 기준):
  - 1인당 하루 10회 대화 × 평균 500 토큰 = 5,000 토큰/일
  - 월 1,000명 × 30일 × 5,000 토큰 = **150M 토큰/월**
  - Input: 75M × $0.150 = **$11.25**
  - Output: 75M × $0.600 = **$45.00**
  - **합계: $56.25/월** (약 75,000원)

#### **GPT-4o** (발음 분석 전용)
- **사용처**: 고품질 발음 분석 및 피드백
- **가격**:
  - Input: $2.50 / 1M tokens
  - Output: $10.00 / 1M tokens
- **예상 사용량**:
  - 1인당 하루 5회 발음 분석 × 200 토큰 = 1,000 토큰/일
  - 월 1,000명 × 30일 × 1,000 토큰 = **30M 토큰/월**
  - Input: 15M × $2.50 = **$37.50**
  - Output: 15M × $10.00 = **$150.00**
  - **합계: $187.50/월** (약 250,000원)

#### **GPT-3.5-turbo** (일반 대화)
- **사용처**: 빠른 일반 대화 (현재 거의 미사용)
- **가격**:
  - Input: $0.50 / 1M tokens
  - Output: $1.50 / 1M tokens
- **예상 비용: $5/월** (거의 미사용)

#### **Whisper-1** (음성 인식)
- **사용처**: 사용자 음성을 텍스트로 변환 (STT)
- **가격**: **$0.006 / 분**
- **예상 사용량**:
  - 1인당 하루 10회 × 평균 20초 = 200초/일 ≈ 3.3분/일
  - 월 1,000명 × 30일 × 3.3분 = **99,000분/월**
  - 99,000분 × $0.006 = **$594/월** (약 790,000원)

#### **TTS-1** (텍스트 → 음성)
- **사용처**: AI 응답을 음성으로 변환
- **가격**: **$15.00 / 1M characters**
- **예상 사용량**:
  - 1인당 하루 10회 × 평균 100자 = 1,000자/일
  - 월 1,000명 × 30일 × 1,000자 = **30M characters/월**
  - 30M × $15.00 = **$450/월** (약 600,000원)
